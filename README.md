# ERP Management System

A web-based ERP Management System developed using Java, Spring Boot, MySQL, HTML, CSS, and JavaScript.

## Features
- User Authentication
- User Management
- Employee Management
- Dashboard
- CRUD Operations
- Database Integration

## Technologies Used
- Java
- Spring Boot
- MySQL
- HTML
- CSS
- JavaScript
- Git & GitHub
